How to Generate a Topology

        
    
            
  1. Define the Trunk: Set the total fiber pairs (FPs) for the main continuous cable.
    2. 
            
  2. Add Branching Units (BUs): Click "+ Add Branching Unit" for every physical BU. Use ▲▼ to reorder and × to delete.
    3. 
            
  3. Assign Destinations (Drops): Inside each BU, add a "Branch Drop" to represent the target station (e.g., CLS 5).
    4. 
            
  4. Map the Fibers: For each drop, specify which FPs route there. Use "+ Add FP Bundle" for non-contiguous pairs.
    5. 
            
  5. Sub-Branching (Optional): Use "+ Add Sub-Branch" to tier the diagram. Sub-branches also support multiple FP bundles.
    6. 
            
  6. Generate: Click "Download .drawio" then open in Draw.io.
