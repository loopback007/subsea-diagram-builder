How to Generate a Topology

  1. Define the Trunk: Set the total fiber pairs (FPs) for the main continuous cable.
  2. Add Branching Units (BUs): Click "+ Add Branching Unit" for every physical BU sitting on the main trunk.
  3. Assign Destinations (Drops): Inside each BU, add a "Branch Drop" to represent the target station (e.g., CLS 5).
  4. Map the Fibers: For each drop, specify which FPs route there. If non-contiguous pairs drop to the same location (e.g., 7-12 AND 19-24), click "+ Add FP Bundle" inside that drop.
  5. Sub-Branching (Optional): If a drop hits a secondary BU before reaching a terminal station, use the "+ Add Sub-Branch" feature to tier the diagram.
  6. Generate: Click "Generate Draw.io Diagram" to download the XML schema, then open it in Draw.io.
