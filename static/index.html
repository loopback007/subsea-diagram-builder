How to Generate a Topology

    
    
        
  1. Define the Trunk: Set the total fiber pairs (FPs) for the main continuous cable.
    2. 
        
  2. FP Line Spacing: Use the spacing slider to increase/decrease the vertical gap between FP lines across the whole diagram (default 20 px).
    3. 
        
  3. Add Branching Units (BUs): Click "+ Add Branching Unit" for every physical BU. Use ▲▼ to reorder and × to delete.
    4. 
        
  4. Set Switch Position: Choose the optical switch state per BU — Default (dot), Position 1 (trunk through), or Position 2 (trunk to branch).
    5. 
        
  5. Assign Destinations (Drops): Inside each BU, add a "Branch Drop" to represent the target station.
    6. 
        
  6. Map the Fibers: For each drop, specify which FPs route there. Use "+ Add FP Bundle" for non-contiguous pairs.
    7. 
        
  7. Sub-Branching (Optional): Use "+ Add Sub-Branch" to tier the diagram.
    8. 
        
  8. Generate: Click "Download .drawio" then open in Draw.io.
